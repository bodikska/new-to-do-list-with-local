# Todo List with LocalStorage

Simple todo list web application built with vanilla JavaScript.

The application allows users to add tasks, remove them, and store the task list in LocalStorage so that tasks remain after the page is reloaded.

## Tech Stack

JavaScript (Vanilla JS)  
HTML  
CSS  
LocalStorage  

## Features

- add new tasks
- delete tasks
- store tasks in LocalStorage
- tasks persist after page reload
- simple and lightweight interface

## Functionality

Users can add tasks using the input field.

Each task is dynamically added to the page using JavaScript and saved to the browser LocalStorage.

When the page is reloaded, the application retrieves the saved tasks from LocalStorage and renders them again.
